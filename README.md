# Kimlyn Pereira Portfolio

A static, responsive portfolio website. No build tools are required.

## Preview locally

Double-click `index.html`, or run a local server from this folder:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a new public GitHub repository, for example `kimlyn-portfolio`.
2. Upload all files and folders from this package to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then click **Save**.
6. GitHub will provide a live address such as `https://YOUR-USERNAME.github.io/kimlyn-portfolio/`.

For a root address such as `https://YOUR-USERNAME.github.io/`, name the repository exactly `YOUR-USERNAME.github.io`.

## Before publishing

- Confirm the LinkedIn URL in `index.html`.
- Confirm whether the telecom project should say **Aircel** or **Airtel**.
- Replace `assets/Kimlyn_Pereira_Resume.docx` whenever the résumé changes.
- Review all metrics and claims for accuracy.
